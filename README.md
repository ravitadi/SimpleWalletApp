
Simple PayPal Wallet App built using ReactJS and Google Material UI.


## To Start
Run `yarn` and `yarn Start`

## To run tests
Run `yarn test`

## Dependencies

- React Scripts
- React
- ReactDOM
- Google Material UI
- React Context
- Kraken JS for end point configuration and security

## Folder Structure

  - UI-components
  - Backend
    - Here the API call would be made
    - Mockdata
